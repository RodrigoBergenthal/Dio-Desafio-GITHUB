##Comandos Windows Terminal

- CLS: LIMPAR
- CD: ENTRAR
- CD..: VOLTAR
- CD /: VOLTAR ORIGEM
- TAB: COMPLETAR
- MKDIR: CRIAR PASTA (DIRETORIO)
- ECHO: MOSTRAR COMANDO - RETORNO
- DEL: DELETAR
- RMDIR: DELETA REPOSITORIO
- LS-a : LISTAR OCULTO
- MV: MOVER

###Comandos terminal GIT

- GIT INIT
- GIT ADD   --- -GIT ADD . (ADICIONA TODOS)
-GIT COMMIT
-GIT STATUS
-GIT REMOT ADD ORIGIN
-GIT PUSH ORIGIN MASTER

- EXEMPLO DE COMANDO PARA COMMIT

- GIT COMMIT -m "Exemplo de descrição"
